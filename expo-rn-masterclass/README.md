## 📖 How to Use This Repo

**Read in order.** Each chapter builds on the previous one. If you're experienced, skip to the chapter you need — every file is self-contained.

**Daily updates** land in `DAILY-UPDATES/` every day. Subscribe to this repo (Watch → Releases) to get notified.

## 🗓️ Daily Updates

A new file is added to `DAILY-UPDATES/` every day. Each contains:

1. **Changelog Watch** — new releases in Expo SDK or React Native
2. **Pattern of the Day** — one pattern, hook, or library worth knowing
3. **Deprecated Today** — anything newly deprecated + its replacement
4. **Speed Tip** — one workflow optimization
5. **Repo to Watch** — one GitHub repo worth starring

## 🤝 Contributing

1. Fork this repo
2. Create a branch: `git checkout -b feature/chapter-name-improvement`
3. Follow the rules: TL;DR block, TypeScript, gotchas section, speed tricks section
4. Submit a PR with a one-line description of what you changed and why

**File rules:**
- Every `.md` opens with a TL;DR block (max 3 bullets)
- All code is TypeScript with `tsx` or `bash` fenced blocks
- Deprecated APIs get a ⚠️ flag with the current replacement
- End every file with a `⚡ Shortcuts & Speed Tricks` section
